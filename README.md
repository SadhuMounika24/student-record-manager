# Student-record-manager
this is simple python based application used to manage student records.
This program allows user to add, view , search, delete, update the details.
this is developed using basic python concepts such as dictionaries, loops and conditional statements.
# Features
. Add new student
. view all students data
. search student by roll no
. delete student data
# Technologies used 
. python 
. dictionary data structured
. loops and conditional statements
# How the program works
The program displays a menu and the user will select an option.
1.Add student
2.View student
3.Search student 
4.Update marks
5.Delete marks
6.Exit
This works by the student data in a key:value dictionary pairs when roll no is entered.
